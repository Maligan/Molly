h1. molly.vim

p. Tiny & beauty GoToFile (Command-T) plugin. Full native Vim solution. 

h2. Getting Started

p. By default it uses <leader>t, but you can change it to whatever you want.

h2. Mappings

* Leader + t  —  open molly
* Enter       —  go to file
* Ctrl + v    —  open in a vertical split window
* Ctrl + s    —  open in a split window
* Ctrl + t    —  open in a new tab
* Up          —  go up
* Ctrl + k    —  go up
* Down        —  go down
* Ctrl + j    —  go down
